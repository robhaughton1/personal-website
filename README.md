# Personal Website

This project is a simple personal website built using HTML and CSS, originally created as an early web development assignment and demonstrates the fundamentals of structuring a webpage, applying custom styling, and organizing assets.

## Features
- **Basic webpage layout** using semantic HTML
- **External CSS stylesheet** for styling and layout
- **Image folder** for organized asset management
- **Clean, readable structure** suitable for beginners

## What I practiced
- Building a webpage from scratch with HTML
- Styling elements using CSS selectors and properties
- Organizing project files (HTML, CSS, images)
- Creating a simple personal or portfolio-style homepage

## Technologies Used
- HTML5
- CSS3

## How to View
Open `index.html` in any web browser to view the website.
